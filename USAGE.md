## 🎯 应用功能导览

### 页面一：📝 **填报分析**（默认）
> 6步渐进式问卷，生成详细志愿报告

**第1步 - 基本信息**
- 姓名（可不填）
- 性别
- 所在省份（主要支持广东）
- 选科组合（物理/历史 + 三科组合）

**第2步 - 成绩信息**
- 🏆 高考成绩：直接填入高考分数
- 📝 模拟考成绩：输入一模/二模/三模分数 → 自动换算预估高考分数

**第3步 - 兴趣与性格**
- 选择感兴趣的方向（16项，可多选）
- 填写目标专业方向（师范、数学、生物等）

**第4步 - 城市选择**
- 勾选意向城市
- "留在广东""就近最好""不限城市" 等快速选项

**第5步 - 家庭资源与其他**
- 家庭职业背景（医、法、金、教等）
- 对离家读书的态度
- 补充说明

**第6步 - 信息确认**
- 检查所有填写信息
- 点击"生成我的志愿填报报告"

↓ **生成报告** ↓

### 页面二：📊 **考试换算**
> 将一模/二模/三模成绩转换为高考预估分数

**操作步骤**
1. 选择考试类型（深圳一模 / 广州一模 / 百日联考）
2. 选择科类（物理类）
3. 输入模拟考分数
4. 即时显示预估高考分数区间

**示例**
- 深圳一模 620分 → 预估高考 620-628分 → 录取概率参考

### 页面三：📋 **查看报告**（点击后出现）
> 完整的志愿填报分析报告

**报告内容**（6大部分）

#### ① 📍 分数定位与位次分析
- 全省位次（约第几名）
- 超越全省考生比例
- 院校竞争层次判断

#### ② 🧠 兴趣性格分析
- 霍兰德职业兴趣模型
- 六维度兴趣分布（雷达图）
- 推荐职业方向

#### ③ 📚 专业方向推荐
- 目标专业匹配度
- 兴趣推荐（基于霍兰德类型）
- 行业趋势热门专业
- 每个专业都标注就业前景

#### ④ 🏛️ 冲稳保院校推荐
**三类方案，可切换查看**

- **🚀 冲刺**（3-5所）
  - 录取概率 30-55%
  - 目标院校比自己分数线更高
  - 需要发挥稳定，务必服从调剂

- **✅ 稳妥**（4-6所）
  - 录取概率 60-85%
  - 分数基本匹配
  - 最保险的选择

- **🛡️ 保底**（3-5所）
  - 录取概率 90%+
  - 明显低于自己分数
  - 确保能被录取

**每所院校展示**
- 院校名称 + 层级（985/211/师范）
- 所属城市和位置
- 专业组代码 + 包含专业
- 2025年实际录取线
- 与考生分数的差值
- 录取概率评估
- 学校特色和政策说明

#### ⑤ 🏙️ 城市选择策略
- 推荐城市排序（按优先级）
- 每个城市的优势分析
- 该城市有哪些推荐院校
- 就业率和薪资参考

#### ⑥ 💡 核心建议
- **⭐ 推荐路径**：冲 → 稳 → 保 的具体院校
- **关键要点**
  - 位次是核心
  - 服从调剂很重要
  - 查五年趋势防大小年
  - 保底必须留
- **⚠️ 风险提示**（6项）
  - 服从调剂提醒
  - 五年趋势查看
  - 体检限报注意
  - 保底志愿要求
  - 师范提前批机会
  - 数据核实提示

## 📱 使用流程示例

### 场景：深圳一模后，想快速定位

**操作步骤**
1. 点击顶部 **📊 考试换算**
2. 选择"深圳一模"
3. 输入一模分数（如620分）
4. 查看预估高考分数（约620-628分）
5. 点击 **📝 填报分析**
6. 用预估分数重新填写问卷
7. 最后会看到根据这个成绩的院校推荐

### 场景：拿到高考分数后，要生成最终报告

**操作步骤**
1. 点击顶部 **📝 填报分析**
2. 走完6步问卷（5分钟左右）
3. 点击"生成我的志愿填报报告"
4. 查看详细报告
5. 对冲稳保院校组合满意？
   - 是 → 保存/打印 PDF 作为填报参考
   - 否 → 点"重新填写"调整条件

## 💡 核心数据解读

### 位次
- **最重要的指标**
- 跨年比较时，用位次而非分数
- 本系统位次基于广东2025年官方一分一段表

### 录取概率
- **冲**：30-55%，拼一拼
- **稳**：60-85%，基本稳妥
- **保**：90%+，放心选

### 分数差
- 正数 = 你的分数高于院校线
- 负数 = 你的分数低于院校线（冲刺时常见）
- 差值越小越冒险

## 🔍 核心功能特色

### ① 一模/二模换算
- 支持深圳一模、广州一模、百日联考
- 基于近5年历年数据统计
- 给出换算区间而不是死的分数（更科学）
- 标注换算参考价值（高分段更准确）

### ② 霍兰德兴趣匹配
- 问卷后根据你的选择自动匹配职业兴趣类型
- 生成雷达图可视化你的兴趣特征
- 推荐最适合你的专业方向

### ③ 冲稳保智能计算
- 算法基于位次差而非分数差
- 自动对比你的成绩和院校往年录取线
- 智能归类到冲/稳/保三档
- 支持切换查看三类院校

### ④ 城市策略分析
- 考虑产业、就业、生活等多维度
- 每个城市标注代表院校
- 就业率和薪资参考
- 根据你的偏好动态推荐

### ⑤ 专业与行业趋势
- 每个专业标注行业趋势（🔥热门/📊稳定）
- 包含国家十五五战略方向
- 新兴行业（AI、低空经济、新能源等）
- 就业出口评估

## 📞 常见问题

**Q: 为什么我的分数和一模位次差这么多？**
A: 一模通常比高考难，且每年难度波动较大。换算结果仅供参考，建议结合多次考试综合评估。

**Q: 保底志愿要填多少个？**
A: 建议至少3-4个，确保90%以上录取概率。宁可保底高一些，也别冒风险。

**Q: 为什么有些高校没有显示？**
A: 系统收录了60+所主要高校，省内和全国重点院校基本覆盖。如需查找其他院校，可用掌上高考APP补充。

**Q: 专业调剂会影响录取概率吗？**
A: 会的。不服从调剂会增加风险，建议冲刺院校必须勾选，稳妥和保底可根据专业意愿选择。

**Q: 能导出做成WORD或PDF吗？**
A: 支持！报告页面有"打印"按钮，可以：
- 用浏览器打印预览（Ctrl+P / Cmd+P）
- 另存为 PDF
- 打印为实体纸张
