## ❓ 常见问题

### Q1: 我的成绩是600分，怎么办？
**A**: 你的位次可能是16000+，超出了本方案的基准范围(2000-15000)。
- 冲刺梯度：只选1-6号(华科/复旦/南大/西交/电科/华南理工)
- 稳妥梯度：改为9-22号(删除前面虚高的)
- 保底梯度：23-30号全部填上
- 建议重新生成针对16000+位次的方案

### Q2: 我特别想去某个学校怎么办？
**A**: 
1. 如果这个学校在冲刺梯度(1-8号)，直接保留
2. 如果这个学校在保底梯度，可以提前到稳妥梯度的后几个位置
3. 如果这个学校不在30校中，可以用位次相近的学校替换

### Q3: 可以不按照这个顺序填吗？
**A**: 可以，但要注意：
1. ✅ 冲刺梯度(1-8号)必须按照位次从低到高排列
2. ✅ 每个梯度内部要呈阶梯式下降
3. ✅ 冲/稳/保的分界线要清楚
4. ❌ 不能"冲-保-稳"交叉混乱

### Q4: 这个方案多久需要更新？
**A**: 
- 🔄 如果4月有二轮模拟，位次变化>1000名，需要调整
- 🔄 如果接近填报，发布新的招生数据，需要验证
- 📌 如果政策没变，位次没变，基本不需要更新

### Q5: 我的位次波动很大，怎么办？
**A**:
- 用**最低位次**作为"保守估计"
- 用**平均位次**作为"标准估计"
- 用**最高位次**作为"乐观估计"
- **按照平均位次填报**，冲刺梯度稍降低预期

## 🎓 最后的建议

### 三个"必做"
1. ✅ **确认位次** - 用3次模拟的平均位次
2. ✅ **在稳妥梯度选最想的** - 这决定最终去向
3. ✅ **检查梯度** - 位次必须呈阶梯式下降

### 三个"必勾"
1. ✅ **冲刺梯度勾"服从调剂"** - 否则被退档风险大
2. ✅ **稳妥梯度建议勾** - 保险但可选
3. ✅ **保底梯度可选** - 已经很安全了

### 三个"必避"
1. ❌ **冲刺不勾服从** - 高风险退档
2. ❌ **稳妥都选"不想去的"** - 可能真的被录
3. ❌ **保底校位次太高** - 安全边际不足
