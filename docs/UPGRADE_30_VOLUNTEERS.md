### 2️⃣ **推荐算法升级：16个志愿 → 30个志愿**

#### 志愿分配策略：
```
【冲刺梯度】志愿1-10（30-50%录取概率）
  - 北京清华、浙大、武大等C9/985校
  - 需要发挥稳定或稍有优势
  - 务必填满6个专业，服从调剂

【稳妥梯度】志愿11-22（60-80%录取概率）
  - 211校、985较弱专业、地方985
  - 分数基本匹配
  - 平衡学校层次与专业匹配度

【保底梯度】志愿23-30（90%+录取概率）
  - 强势一本、地方重点
  - 确保录取
  - 可适当填报热门专业
```

#### 代码改动：
```javascript
// analyzer.js - 第305-307行
return {
  chong: chong.sort(sortFn).slice(0, 10),  // 冲：5个 → 10个
  wen: wen.sort(sortFn).slice(0, 12),      // 稳：6个 → 12个
  bao: bao.sort(sortFn).slice(0, 8),       // 保：5个 → 8个
};
// 总计：16个 → 30个
```

### 3️⃣ **表格布局优化**

#### 表格新列：
- 志愿号：1-30（全新的排号方式）
- 院校名称：30所不同高校
- 专业组：对应专业组编号
- 最低分 & 历年数据：3年录取数据
- 录取概率：基于分差计算
- 推荐专业：根据专业组细分
- **💰 学费备注**（已有）：每所校每组的学费说明
- 填报策略：冲/稳/保三类不同建议

#### 表格行数：30行（从原来的10行扩展）

---

### 4️⃣ **填报策略调整**

**原三梯度**：
```
冲(1-3) + 稳(4-8) + 保(9-10) + 垫(11) + 捡(12) = 12个
```

**新三梯度**：
```
冲(1-10) + 稳(11-22) + 保(23-30) = 30个
```

**优势**：
- ✅ 更细粒度的分层
- ✅ 冲刺机会更多（10个vs 3个）
- ✅ 保底选择更充分（8个vs 2个）
- ✅ 稳妥过渡更平滑（12个vs 5个）

## 🚀 使用指南

### 学生如何操作

1. **填完7个步骤** → 点击"生成报告"
2. **报告展示** → 自动推荐30个志愿
3. **表格查看** → 按冲稳保三梯度查看
4. **按序填报** → 冲1-10 → 稳11-22 → 保23-30

### 关键提示

- 📌 冲刺志愿**务必填满6个专业**，勾选服从调剂
- 📌 稳妥志愿**兼顾分数与专业**，适度调剂
- 📌 保底志愿**确保录取**，可选热门
- 📌 每个志愿内**院校可自主调序**
- 📌 参考**学费备注**选择中外合作专业
