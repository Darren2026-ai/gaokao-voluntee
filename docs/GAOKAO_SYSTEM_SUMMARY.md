## 💎 核心算法与创新

### 1️⃣ 智能位次计算
- 基于广东2025年官方一分一段表
- 线性插值精确到分数点
- 支持用户手动输入位次覆盖

### 2️⃣ 冲稳保自动分类
```
冲 = 目标院校位次 < 学生位次 (差值1000-5000)
稳 = 位次接近 (差值±1000)
保 = 目标院校位次 > 学生位次 (差值2000-8000)
```

### 3️⃣ 一模/大型考试换算
- 深圳一模：×1.02 难度系数
- 广州一模：×1.01 难度系数
- 百日联考：×1.015 难度系数
- 基于5年历年数据建立转换表

### 4️⃣ 霍兰德兴趣模型集成
- 六维度兴趣评分
- 关键词自动匹配
- 雷达图可视化
- 专业方向推荐

### 5️⃣ 城市匹配策略
- 产业链考虑
- 就业资源评估
- 生活成本对比
- 优先级排序
