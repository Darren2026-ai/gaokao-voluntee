# 高考志愿填报 - AI深度分析与务实报告生成系统

## 📊 核心改进原则

### 一、"不浪费志愿"的科学定义

#### 位次梯度设置规范

**现状**：620-660分 → 全省位次2000-15000名

| 梯度层级 | 位次范围 | 录取概率 | 志愿号 | 核心原则 |
|---------|---------|--------|--------|---------|
| 🚀 冲刺高 | 你的位次 -500 到 -1000 | 20-30% | 1-3 | **绝不能冲清北等顶尖校**，要选"有概率的冲" |
| 🎯 冲刺中 | 你的位次 -200 到 -500 | 30-45% | 4-6 | 选择往年有录取先例的学校 |
| 🎪 冲刺低 | 你的位次 +200 到 -200 | 45-60% | 7-10 | 基本不浪费，风险可控 |
| ✅ 稳妥中等 | 你的位次 +200 到 +1500 | 65-85% | 11-20 | **核心梯度，最容易录取的范围** |
| 🛡️ 保底高 | 你的位次 +1500 到 +3000 | 90-95% | 21-28 | 保证录取的关键范围 |
| 🔐 保底低 | 你的位次 +3000 到 +5000 | 98%+ | 29-30 | **最后1-2所学校，绝对保险** |

---

### 二、"实事求是冲高"的标准

#### ❌ 错误做法（浪费志愿）
- 620分考生报清北(位次20-150) → **0%录取概率，100%浪费**
- 8000名考生报浙大(位次600) → **虚高2000名以上，99%浪费**
- 冲刺梯度全是"梦想校" → **冲得太高，没有过渡，容易全落空**

#### ✅ 正确做法（科学冲高）
- 基准位次8000名 → 冲到位次**6500-7500**的学校（下降500-1500名，有先例录取）
- 比如：华南理工(往年位次6800)、西交大(往年位次4200) → **都有相近位次考生录取的先例**
- 冲梯度设置：位次6500/7000/7200/7500/8000/8200 → **逐步下降，形成阶梯**

---

### 三、基于真实数据的分析框架

#### 第一步：收集数据
```
需要的关键数据：
1. 目标学校的2023/2024/2025年录取位次（至少3年）
2. 位次变化趋势（上升/下降/波动）
3. 同梯度其他学校的位次对标
4. 该学校是否扩招或缩招
```

#### 第二步：AI分析
```
分析维度：
1. 位次稳定性评分（变化±500以内 = 稳定；变化±1000 = 波动）
2. 与考生位次的匹配度（距离、方向、风险等级）
3. 梯度合理性检查（是否形成阶梯，有无跳跃）
4. 浪费程度评估（位次差距过大的学校标记为"高浪费")
```

#### 第三步：生成报告
```
输出内容：
- 分析数据表（每所学校的历史位次、稳定性、风险）
- 梯度评估（冲得是否过高，保得是否过低）
- 具体改善建议（哪些学校需要替换，为什么）
- 优化后的30所学校方案
```
