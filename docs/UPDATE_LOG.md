# 高考志愿填报系统 - 选科组合优化更新

**更新日期**：2026年3月23日  
**更新版本**：V2.1

---

## 📝 更新内容

### 1. **第一步 - 基本信息页面**

#### 优化前
- 选科组合显示为简单的 `<option>` 标签
- 历史类只有一个"历史类"选项，没有具体的选科组合

#### 优化后
- 使用 `<optgroup>` 分组展示物理类和历史类
- 物理类包含：物化生、物化政、物化地、物生地、其他组合
- 历史类包含：**史生地、史政地**（两个具体选科组合）
- 每个选项都有清晰的组合标签，帮助考生快速定位

```html
<optgroup label="🔬 物理类">
  <option value="物理类（物化生）">物理类（物化生）- 理科经典组合</option>
  <option value="物理类（物化政）">物理类（物化政）- 文理兼修</option>
  <option value="物理类（物化地）">物理类（物化地）- 文理兼修</option>
  <option value="物理类（物生地）">物理类（物生地）- 文理兼修</option>
  <option value="物理类（其他组合）">物理类（其他组合）</option>
</optgroup>
<optgroup label="📖 历史类">
  <option value="历史类（史生地）">历史类（史生地）</option>
  <option value="历史类（史政地）">历史类（史政地）</option>
</optgroup>
```

---

### 2. **代码层面的优化**

#### 2.1 js/app.js - 新增选科映射表
```javascript
const SUBJECT_TYPES = {
  '物理类（物化生）': { type: '物理类', combination: '物化生', label: '物理类 · 物化生' },
  '物理类（物化政）': { type: '物理类', combination: '物化政', label: '物理类 · 物化政' },
  '物理类（物化地）': { type: '物理类', combination: '物化地', label: '物理类 · 物化地' },
  '物理类（物生地）': { type: '物理类', combination: '物生地', label: '物理类 · 物生地' },
  '物理类（其他组合）': { type: '物理类', combination: '其他', label: '物理类 · 其他' },
  '历史类（史生地）': { type: '历史类', combination: '史生地', label: '历史类 · 史生地' },
  '历史类（史政地）': { type: '历史类', combination: '史政地', label: '历史类 · 史政地' },
};
```

**作用**：
- 提供选科组合的完整映射
- 支持后续扩展新的选科组合
- 便于在报告中格式化显示选科信息

#### 2.2 新增 formatSubjectDisplay() 函数
```javascript
function formatSubjectDisplay(subject) {
  const mapping = SUBJECT_TYPES[subject];
  if (mapping) {
    return mapping.label;
  }
  return subject;
}
```

**作用**：
- 统一格式化选科显示
- 在确认信息和报告中保持一致的显示格式

### 3. **确认页面（第7步）的改进**

- 选科显示从原始值改为格式化的标签
- 示例：`物理类（物化生）` → `物理类 · 物化生`

---

### 4. **志愿分析报告的改进**

#### 4.1 报告头部信息 (report-meta)
- **新增**：在报告最上方显示"选科组合"卡片
- 与成绩、位次、成绩层次等信息并排展示
- 帮助考生快速了解自己的选科背景

#### 4.2 分数定位部分
- 位次标签动态显示：`全省位次（物理类）`或`全省位次（历史类）`
- 根据用户选科组合自动调整

#### 4.3 历史类考生的特殊提示
- 在位次提醒下方，为历史类考生添加了额外的提示框
- 内容：提醒考生参照招生章程确认具体招生计划
- 背景色：黄色（#FEF3C7），便于识别

```html
${user.subject.includes('历史') ? `
<div style="background:#FEF3C7; border-radius:8px; padding:14px 18px; margin-top:12px; font-size:14px; color:#92400E;">
  📌 <strong>历史类考生提示：</strong>当前推荐的院校数据以物理类为主。由于历史类考生数量及录取数据有所不同，建议在查看院校时，参照各校的招生章程确认具体的招生计划和分数要求。
</div>
` : ''}
```

---

### 5. **考试换算页面的优化**

- 历史类选项从"历史类（暂不支持）"改为"历史类（参考物理类换算）"
- 给予用户更清晰的说明，说明换算时的参考依据
