🔍 浏览器控制台测试指南

第一步：打开浏览器控制台

  • Chrome/Edge/Firefox: 按 F12 或 Ctrl+Shift+J (Windows) / Cmd+Option+J (Mac)
  • Safari: 右键 → 检查元素 → 控制台

第二步：访问原始页面

第三步：在控制台中复制粘贴以下代码

测试 1: 检查基本依赖

console.log('COLLEGES:', typeof COLLEGES, COLLEGES?.length);
console.log('GaoKaoAnalyzer:', typeof GaoKaoAnalyzer);
console.log('generateReport:', typeof generateReport);
console.log('ExamConverter:', typeof ExamConverter);

测试 2: 检查按钮是否存在

const btn = document.querySelector('.btn-submit');
console.log('按钮元素:', btn);
console.log('按钮 onclick:', btn?.onclick);

测试 3: 手动触发生成报告

// 先填写必要的表单字段
document.getElementById('f-score').value = 630;
document.getElementById('f-name').value = '测试';
document.getElementById('f-province').value = '广东';
document.getElementById('f-subject').value = '物理类（物化生）';

// 然后手动调用生成报告
console.log('正在生成报告...');
generateReport().then(() => {
  console.log('✅ 报告生成完成！');
}).catch(err => {
  console.error('❌ 生成失败:', err.message);
  console.error('完整错误:', err);
});

测试 4: 检查分析器是否工作

try {
  const analyzer = new GaoKaoAnalyzer({
    score: 630,
    interests: [],
    cityPreference: []
  });
  const result = analyzer.analyze();
  console.log('✅ 分析器工作正常');
  console.log('结果:', result);
} catch (e) {
  console.error('❌ 分析器错误:', e.message);
}
