## 🌟 系统简介

本系统是一个基于 **2025 年广东省真实录取数据** 的高考志愿智能分析工具，采用 **六维度分析框架**，为考生提供精准的志愿填报建议。

### 六大分析维度

1. **兴趣与性格** (35%) - 霍兰德职业兴趣 + 专业大类
2. **成绩定位** (25%) - 分数、位次、冲稳保分析
3. **城市选择** (20%) - 一线/新一线城市就业前景
4. **专业推荐** (10%) - 国家战略产业匹配
5. **院校层次** (5%) - 985/211/双一流分类
6. **家庭资源** (5%) - 个人背景优化


## 📝 详细填报流程

### Step 1️⃣: 基本信息

**必填项**:
- 📱 **高考分数** - 2025年高考总分或模拟考分数
- 🗺️ **所在省份** - 默认广东，可选其他省份
- 📚 **选科组合** - 物理类/历史类

**选填项**:
- 👤 姓名 - 便于生成个性化报告
- ⚧️ 性别 - 某些专业可能有性别偏好

### Step 2️⃣: 成绩信息

**两种方式二选一**:

#### 方式A: 填写高考成绩 (推荐)
- 📊 **高考总分** - 请输入 200-750 分
- 📈 **全省位次** (选填) - 更精准的定位

#### 方式B: 填写模拟考成绩
- 📋 考试类型 - 深圳一模/广州一模等
- 📊 分数 - 模拟考总分
- 🏆 全市名次 (选填) - 帮助换算全省位次
- 📈 全市考生总数 (选填) - 计算百分位

**系统会自动**:
- ✅ 估算全省位次
- ✅ 判断分数层次 (顶尖/985/211/一本等)
- ✅ 给出冲稳保分析

### Step 3️⃣: 兴趣与性格

#### 第一部分: 霍兰德职业兴趣
选择符合你的职业类型 (可多选):
- 🔬 科学探究 - 适合理工科研
- 💻 编程技术 - 计算机/IT
- 📐 数学逻辑 - 数学/金融/数据
- 🧬 生命科学 - 生物/医学
- ... (共16个选项)

#### 第二部分: 专业大类
直接选择感兴趣的专业方向 (可多选，⭐标红为国家重点):
- 💻 计算机类 - 最热门，就业好
- ⚡ 电子信息类 - 集成电路方向
- 🤖 人工智能类 ⭐ - 新兴热点
- ⚙️ 机械工程类 - 传统工程强校
- ... (共20个专业大类)

### Step 4️⃣: 城市选择

选择意向城市 (可多选):

**一线城市** (资源最多，竞争最大):
- 🏙️ 北京 - 全国中心
- 🏙️ 上海 - 国际都市
- 🏙️ 深圳 - 湾区核心
- 🏙️ 广州 - 华南中心

**新一线城市** (性价比高):
- 杭州 - 互联网之城
- 武汉 - 性价比之王
- 成都 - 新兴产业

**地方选项**:
- 留在广东 - 省内均可
- 就近最好 - 家庭建议

### Step 5️⃣: 志愿优先级

根据你的目标选择一个优先级:

| 选项 | 特点 | 适合人群 |
|------|------|---------|
| 🏙️ **城市优先** | 优先一线/新一线城市 | 重视平台、视野、实习机会 |
| 🎓 **学校优先** | 优先985/211名校 | 看重学历、校友资源、深造 |
| 📚 **专业优先** | 优先强势学科 | 明确职业方向、行业需求 |
| ⚖️ **平衡选择** | 综合考虑三因素 | **首次填报推荐** |

💡 **建议**: 不确定时选"平衡选择"，风险最低

### Step 6️⃣: 家庭与资源

**家庭职业背景** (可多选):
- ⚕️ 医疗资源 - 有利于医学专业
- ⚖️ 法律资源 - 有利于法学专业
- 💰 金融资源 - 有利于金融专业
- ... (其他资源或无)

**离家态度**:
- 支持 - 全国任意城市
- 省内 - 优先广东省内
- 市内 - 优先深圳/广州附近

**补充说明** (可选):
- 特殊情况说明
- 专业禁忌或偏好
- 其他重要信息

### Step 7️⃣: 确认提交

检查已填信息，确认无误后点击:
```
🔍 生成我的志愿填报报告
```

系统会在 **3-5 秒内** 生成完整报告。

**填报建议**: 按冲-稳-保-垫-捡顺序依次填报

- 城市优先级排序
- 城市就业率数据

## 💡 常见问题

### Q1: 我只有模拟考成绩，可以用吗?
**A**: 可以的！系统会根据历年数据自动换算为高考成绩。建议用多次模拟考的平均分，更准确。

### Q2: 591 分能上 985/211 吗?
**A**: 可以的！591 分在广东大约排名 11500 名左右，属于 **211 优质段**，可冲击部分 985 的较弱专业。具体见冲稳保方案。

### Q3: 报告中的分数数据准不准?
**A**: 数据基于 **2025 年广东省官方录取数据**，但注意:
- 不同年份试题难度不同
- 招生计划可能调整
- **一定要以官方最新数据为准**

### Q4: 可以选择多个优先级吗?
**A**: 不行，需要选一个。但系统会针对你的选择生成对应的院校推荐顺序。

### Q5: 模拟考成绩如何换算?
**A**: 系统根据历年深圳一模、广州一模等与高考分数的对标关系，用线性拟合或统计回归的方法估算。

### Q6: 报告可以导出吗?
**A**: 可以用浏览器的打印功能 (Ctrl+P / Cmd+P) 另存为 PDF。

### Q7: 系统支持其他省份吗?
**A**: 目前主要针对 **广东物理类**。其他省份数据正在开发。

## 🎯 使用建议

### 填报前准备

1. ✅ 了解你的分数和位次
2. ✅ 了解自己的兴趣和职业方向
3. ✅ 研究想去的城市
4. ✅ 咨询家长的想法

### 填报时建议

1. **诚实填写** - 数据越准确，建议越精准
2. **多角度思考** - 不要只看排名，看专业实力
3. **留意红星** - ⭐ 标记的是国家重点扶持产业
4. **平衡选择** - 第一次填报建议选"平衡"

### 填报后行动

1. **核实数据** - 再次确认录取线和招生计划
2. **咨询老师** - 跟学校老师/顾问二次沟通
3. **研究院校** - 了解各校强势学科
4. **关注调剂** - 考虑是否同意专业调剂
