## 🚨 3个最常见的错误

### ❌ 错误1：冲刺梯度填太多"梦想校"
```
错误：1-5号都是清北浙大复旦南大
理由：虚高冲刺，位次跳跃太大
改正：只在冲刺梯度放1-2个"梦想校"，其他选位次接近的
```

### ❌ 错误2：稳妥梯度填不了满意的校
```
错误：稳妥梯度都是"不太想去的学校"
理由：可能最后被这些校录取
改正：在稳妥梯度(9-22号)优先选择自己最满意的学校
```

### ❌ 错误3：保底梯度的最后校位次太高
```
错误：最后校的位次只比自己低1000名
理由：一旦前面全落榜，保底校也可能录不了
改正：最后1-2校的位次必须明显低于自己(≥2000名)
```

## 📱 应急快速调整

### 高考发挥超常(+20-30分)
- 删除冲刺梯度的8号(浙大)
- 替换为清华北大试试
- 稳妥梯度可以调强(选更好的校)

### 高考发挥失常(-20-30分)
- 冲刺梯度保留1-3号
- 稳妥梯度改为4-22号(扩大稳妥范围)
- 保底梯度23-30号确保齐全

### 模拟成绩波动大
- 用**历次成绩的平均位次**作为基准
- 按平均位次填报
- 冲刺梯度稍降低预期
