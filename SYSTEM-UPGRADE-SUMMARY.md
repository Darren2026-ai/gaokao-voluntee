**核心思想**：
```
冲刺梯度设定原则：
- 冲刺高：你的位次 -500到-1000（20-30%录取概率）
- 冲刺中：你的位次 -200到-500（30-45%录取概率）
- 冲刺低：你的位次+200到-200（45-60%录取概率）

这样设定，冲得有希望，不浪费志愿
```

## 📈 数据驱动的改进

### 分析方法

每次生成报告时，新系统会自动：

#### 第1步：收集数据
```
✓ 目标学校的2023/2024/2025年录取位次
✓ 位次变化趋势(↑/→/↓)
✓ 同梯度其他学校的对标位次
✓ 该学校扩招/缩招情况
```

#### 第2步：AI分析
```
✓ 位次稳定性评分(±500内=稳定；±1000=波动)
✓ 与考生位次的匹配度分析
✓ 梯度合理性检查(是否形成阶梯)
✓ 浪费程度评估(位次差距定量化)
```

#### 第3步：生成报告
```
✓ 分析数据表(历史位次、稳定性、风险)
✓ 梯度评估(冲是否过高、保是否过低)
✓ 具体改善建议(哪些学校需替换)
✓ 优化后的30所学校方案
```
