# 高考志愿智能填报分析系统

一个本地Web应用，帮助高考生进行多维度志愿填报分析，基于2025年广东省真实录取数据。

## 🎯 核心功能

### 1. **智能问卷系统**（6步）
- **基本信息**：姓名、性别、省份、选科
- **成绩输入**：高考成绩或模拟考成绩智能换算
- **兴趣分析**：16项兴趣选择，匹配霍兰德职业兴趣模型
- **城市选择**：一线、新一线、省内多地选项
- **家庭资源**：医、法、金、教等背景匹配
- **信息确认**：提交前完整预览

### 2. **六维度综合分析**
```
├─ 分数定位（位次、百分位、层次）
├─ 兴趣性格（霍兰德RIASEC模型 + 雷达图）
├─ 专业推荐（热门、兴趣、趋势混合）
├─ 城市策略（就业、资源、宜居性）
├─ 冲稳保院校（基于位次计算）
└─ 风险提示（体检、调剂、大小年等）
```

### 3. **志愿报告**
生成专业志愿填报报告，包含：
- 📊 成绩与位次详细分析
- 🧠 兴趣性格分析（含雷达图）
- 📚 专业方向推荐
- 🏛️ 冲稳保院校组合（可切换）
- 🏙️ 城市选择策略
- 💡 核心建议与填报要点
- ⚠️ 风险提示清单
- 🖨️ 支持打印/导出PDF

### 4. **考试换算工具**
- 深圳一模 → 预估高考分数
- 广州一模 → 预估高考分数
- 百日冲刺联考 → 预估高考分数
- 包含换算区间和参考价值评估

### 5. **完整院校库**
- **60+所院校**（985、211、师范、普通本科）
- **2025年真实录取数据**（分数线、位次、专业组）
- **政策说明**（服从调剂、零调剂不退档等）
- **专业特色**（产业匹配、就业方向）


## 🧠 核心算法

### 1. 位次估算（线性插值）
基于广东2025年物理类一分一段表，精确到分数点。
```javascript
const rank = interpolateRank(score) // 基于一分一段查表
```

### 2. 冲稳保计算
```
冲 = 目标位次 < 学生位次 （差距1000-5000）
稳 = 目标位次基本相等 （差距±1000）
保 = 目标位次 > 学生位次 （差距2000-8000）
```

### 3. 录取概率评估
```
冲：30-55% （根据分数差）
稳：60-85% （分数匹配段）
保：90-95%+ （明显低于学生成绩）
```

### 4. 一模/大型考试换算
基于近5年历年数据，建立换算表：
- 深圳一模难度系数 ×1.02
- 广州一模难度系数 ×1.01
- 支持分数区间预警

## 📋 使用建议

### 最佳时间点
1. **4月（一模后）**：用一模成绩换算，初步定位
2. **5月（二模后）**：完善定位，开始研究院校
3. **6月初（高考前）**：最后确认备选方案
4. **6月中（查分后）**：用实际分数生成最终报告
